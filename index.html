Online Exam System - Demo

Online Examination:

“Online Examination” system is a non removable examination pattern of today’s life. We need more time saving and more accurate examination system as the number of applicants is increasing day by day. For all students and professionals, it is very important to have some basic understanding about the online examination system. On this system you will get source code with the running project. It will help you to understand the concept of the project. This system can be used in schools, colleges, coaching and institutes for checking online exams.

Technologies used
  • Back-End Language: PHP
  • Front-End Languages: HTML,CSS & JavaScript
  • Database : Mysql
  • Storage : local storage

Use Cases

Online Exam will have 3 types of users. First one is Admin, second is Student and third is Teacher. Each one of them has different tasks over the system, as mentioned below :

  • Admin

    The important member of system is admin his/her abilities are below:

    • Can manage every member of the system (student & teacher ).
    • Has the ability to register students and teachers.
    • Has the ability to create exams and updates.
    • Has the ability to accept and reject registered students ,teachers and admins to the system.
    • Has the ability to add and delete colleges , departments and subjects to the system.
    • Has the ability to reset passwords of any member of the system.
  • Student

    is one of member of the system and he/she has capabilities below:

    • Can do a live exam online.
    • Can update some information of the profile such as image,name,email,date of birth and reset password.
    • Can review his/her exams done before.
    • Can see his/her exam scores.
    • Can see how many exams are done of every subject.
  • Teacher(Lecturer)
    • Can register exams to students of one group or all students that are in his/her class.
    • Can add questions to the subject he/she has.
    • Can notify students of the time of exams.
    • Can monitor students during the exam which one is available or absent.
    • Can see all Exams have been done by students.

Limitations of project:

This project is not complete for every use cases and actions. It a web-based application but, it hasn’t notification system such as student has been logged in, student has finished exam or alert student’s exam time. On the other hand, the security is an issue we don’t strong security strategies to put on the internet and be a public system for everyone. It now running locally. System exams scheduled by date and time; student should available at a time to do his/her exam of the subjects they have to exam.
